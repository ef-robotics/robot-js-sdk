# DEV

## Getting Started

Pull the SDK from Autoxing
```
$ npm i -save-dev @autoxing/robot-js-sdk
```

Compile
```
$ tsc
```

Execute
```
$ node dist/main.js
```

## Get latest RGB image
connet to robot's local hotspot network, password: ```12345678```
```
$ ./get_image.sh
```
